# 駕照日文譯本 線上申請 SOP

> 日本租車自駕需要：護照 + 台灣駕照 + 日文譯本（三樣缺一不可）
> 申請費用：NT$128（規費 100 + 郵資 28）+ 信用卡手續費 20 = **NT$148**
> 作業時間：**7-12 個工作天**郵寄到家
> 不用正本、不用跑監理站

## 申請步驟

### 步驟 0：註冊會員（已有帳號跳過）

1. 打開 [監理服務網](https://www.mvdis.gov.tw)
2. 右上角「加入會員」
3. 用**自然人憑證**或**健保卡**驗證身分
4. 設定帳號密碼

### 步驟 1：登入 → 選擇駕照

1. 打開 [駕照日文譯本申請](https://www.mvdis.gov.tw/m3-emv-drv/driver/jpApply)
2. 會跳到**會員登入**頁面 → 輸入帳號密碼登入
3. 進入「駕照日文譯本」頁面
4. 看到「可以申請的駕照種類」
5. 選擇**汽車** → 按「申請譯本」

### 步驟 2：確認日文譯本資訊

1. 系統自動帶出你的資料：
   - 駕照姓名
   - 出生年月日
   - 特許車種（普通小型車）
   - 駕照管轄編號
   - 駕照有效期限
   - 駕籍地址
2. 選擇取件方式：**郵寄**
3. 確認郵寄地址和電話
4. 按「確認資料無誤」
5. 跳出視窗確認駕籍地址 → 按「確認駕籍地址無誤」

### 步驟 3：繳納費用

1. 繳費明細：
   - 駕照日文譯本：100 元
   - 郵資：28 元
   - 信用卡手續費：20 元
   - **合計：148 元**
2. 選擇「信用卡繳費」
3. 輸入信用卡卡號（1234-1234-1234-1234 格式）
4. 輸入有效月年（四碼，格式 MMYY）
5. 按「確定金融帳號資訊」

### 完成

- 看到「繳費成功！」畫面就完成了
- 約 7-12 個工作天會掛號寄到你填的地址
- 繳費後無法變更申請、無法退費

---

## 注意事項

- 每個人要用**自己的帳號**登入申請，無法線上代辦
- 駕照有效期限顯示「免換照」就沒問題
- 如果駕照有印有效期限，只要未過期就能在日本使用
- 日文譯本效期跟台灣駕照相同
